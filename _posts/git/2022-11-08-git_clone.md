## 설명

깃허브에 있는 파일 로컬에 다운받아 작업하기



# 방법

```
1. git init 
2. git clone "깃허브에서 내려받을 주소"
3. cd 내려받은 폴더 접속
4. 내용 수정 및 추가
5. git add .
6. git commit -m '코멘트'
7. git push
```

## ❗마치며

간단한 내용이지만 다시 한번 기억하자는 의미에서 작성한다.
