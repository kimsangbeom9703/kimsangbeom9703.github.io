## 이 게시글은 로컬에서 블로그 작성 시 로컬 서버 구동하는 법을 자주 잊어먹어서 포스팅한다..!


# 방법

```
1. git pull 
2. bundle exec jekyll serve 
```

# 풀이

```
1. 로컬과 git server 의 블로그 파일 구성이 다를 경우 다운받는다.
2. http://127.0.0.1:4000 에서 확인이 가능하다.
```
